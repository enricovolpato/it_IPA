# Dizionario IPA

Fast, AI-written implementation of espeak-ng to Italian phonological grammar, with postprocessing to include structures that span words.

Site: https://YOUR_USERNAME.github.io/dizionario/

## Links

- [PHONOLOGICAL_RULES.md](PHONOLOGICAL_RULES.md)

## Project Outline

- Web app that converts Italian text to IPA in the browser
- eSpeak-ng (WASM) base transcription
- Italian phonological postprocessor for cross-word rules
- Vite + TypeScript frontend
- GitHub Pages deployment workflow

## Development

```bash
npm install
npm run dev
```
